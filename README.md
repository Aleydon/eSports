#### :arrow_forward: Get Started:

1. Clone this repo

```sh
git clone https://github.com/Aleydon/eSports.git
```

2. Install NPM packages

```sh
npm install or yarn install
```

3.  Run this project

```sh
npm run dev or yarn dev
```
